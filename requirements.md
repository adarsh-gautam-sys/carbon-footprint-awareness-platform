# PromptWars 2026 Challenge Requirements

## Challenge 3

Carbon Footprint Awareness Platform: design a solution that helps individuals understand, track, and reduce their carbon footprint through simple actions and personalized insights.

## Scoring-Aligned Requirements

1. Define a real-world problem, target users, and why AI is needed.
2. Build an AI agent that understands input, breaks tasks down, selects tools, executes actions, and returns structured output.
3. Include frontend, backend/API, LLM, tool execution, storage, and deployment layers.
4. Use Google services effectively, especially Gemini and Cloud Run.
5. Keep code modular, secure, efficient, tested, accessible, and aligned to the challenge.
6. Document tool usage, prompt engineering, architecture, validation, and human vs AI responsibilities.

The original local guide is preserved in `requirements_guide.md`; implementation evidence is documented in `docs/`.
